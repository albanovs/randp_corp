Регистрация ОсОО в Кыргызской Республике для иностранных граждан

Информационное приложение

Данное приложение предназначено для иностранных граждан (нерезидентов), планирующих регистрацию компании в форме ОсОО (Общество с ограниченной ответственностью) на территории Кыргызской Республики.

1. Кто может зарегистрировать ОсОО

  • быть учредителем ОсОО (допускается 100% доля);
  • быть учредителем и директором ОсОО;
  • быть учредителем, назначив директором иное лицо.
Ограничений по гражданству для регистрации ОсОО не предусмотрено.

2. Общая суть процедуры

1. Подготовка документов
2. Государственная регистрация компании
3. Постановка на налоговый учёт
4. Изготовление печати
5. Открытие расчётного счёта
6. Получение лицензий (при необходимости)

Большинство процедур может быть выполнено по нотариальной доверенности, без личного присутствия иностранного гражданина.

3. Документы от иностранного гражданина

Обязательные документы

  • Паспорт иностранного гражданина;
  • Нотариально заверенный перевод паспорта на русский язык;
  • Контактные данные (телефон, e-mail);
  • Нотариальная доверенность на представителя (регистрация ОсОО, налоговый учёт, получение ПИН, лицензии, представление интересов в госорганах).

Дополнительная информация

  • Виза для регистрации ОсОО не требуется;
  • Персональный идентификационный номер (ПИН):
    • при наличии — предоставляется номер;
    • при отсутствии — может быть получен нами по доверенности.

4. Информация, которую необходимо определить заранее

  • полное и сокращённое наименование компании;
  • виды деятельности;
  • необходимость лицензий или разрешений;
  • размер уставного капитала;
  • исполнительный орган (директор / генеральный директор);
  • юридический адрес;
  • налоговый режим.

5. Государственная регистрация ОсОО

Регистрация осуществляется через электронную систему государственных органов Кыргызской Республики.

  • Свидетельство о государственной регистрации;
  • Выписка о зарегистрированном юридическом лице.

6. Налоговая регистрация

После регистрации ОсОО ставится на учёт в налоговых органах.

  • Регистрационная карточка налогоплательщика.

7. Изготовление печати

Изготавливается корпоративная печать ОсОО.

8. Открытие банковского счёта

Директор ОсОО должен лично присутствовать при открытии счёта.

Основные документы для банка

  • Паспорт директора с нотариальным переводом;
  • Устав ОсОО;
  • Решение учредителя;
  • Приказ о назначении директора;
  • Свидетельство о регистрации и выписка;
  • Регистрационная карточка налогоплательщика;
  • Карточка образцов подписей и печати;
  • Печать ОсОО;
  • Доверенность.

9. Лицензирование (при необходимости)

При лицензируемых видах деятельности мы сопровождаем получение лицензий в уполномоченных государственных органах.

10. Сроки процедуры

  • Регистрация ОсОО — 3–5 рабочих дней;
  • Налоговая регистрация — 1–2 рабочих дня;
  • Открытие счёта — 1–3 рабочих дня;
  • Лицензирование — до 30 рабочих дней.

11. Результат для клиента

  • зарегистрированное ОсОО;
  • налоговый учёт;
  • корпоративная печать;
  • банковский счёт;
  • лицензии (при необходимости);
  • полный пакет учредительных документов.

12. Дополнительные положения

  • 100% иностранное владение допускается;
  • минимальный уставный капитал не установлен;
  • возможна дистанционная регистрация по доверенности;
  • личное присутствие требуется только в банке.
